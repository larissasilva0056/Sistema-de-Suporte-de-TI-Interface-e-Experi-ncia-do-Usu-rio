Sistema de Suporte de TI – Interface e Experiência do Usuário

Este projeto apresenta o desenvolvimento de um protótipo de sistema de suporte técnico, criado no Figma, com foco na construção de uma interface intuitiva e na definição de uma jornada do usuário eficiente.

Link do protótipo:
[https://www.figma.com/design/026wgVcpegUqAims2DRSmg/Projeto-de-suporte-de-ti](https://www.figma.com/design/026wgVcpegUqAims2DRSmg/Projeto-de-suporte-de-ti)

Sobre o Projeto

O sistema foi idealizado para facilitar o atendimento de suporte técnico, permitindo que usuários registrem, acompanhem e finalizem chamados de forma simples e organizada.

A proposta busca resolver problemas comuns em ambientes corporativos, como falta de organização, demora no atendimento e dificuldade de comunicação entre usuário e equipe de suporte.

 Design e Prototipação

O protótipo foi desenvolvido utilizando o Figma, ferramenta amplamente utilizada para criação de interfaces e protótipos.

O design prioriza:

* Interface limpa e moderna
* Navegação simples e intuitiva
* Padronização de elementos visuais
* Experiência centrada no usuário

 Funcionalidades Principais

* Abertura de chamados de suporte
* Visualização de solicitações
* Acompanhamento de status
* Organização por prioridade
* Interface de fácil uso para usuários e técnicos

 Jornada do Usuário

O fluxo do sistema foi estruturado para ser direto e eficiente:

1. Acesso ao sistema
2. Criação de chamado
3. Análise pela equipe de suporte
4. Atendimento da solicitação
5. Finalização do chamado

Objetivo do Projeto

Desenvolver uma solução que melhore a experiência do usuário no suporte técnico, tornando o processo mais ágil, transparente e organizado.

 Tecnologias e Ferramentas

* Figma (UI/UX Design)
* Conceitos de UX/UI
* Modelagem de jornada do usuário

 Resultados Esperados

* Redução no tempo de atendimento
* Melhor organização de chamados
* Aumento da satisfação do usuário
* Padronização do processo de suporte
